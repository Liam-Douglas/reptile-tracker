About Backup & Restore

What's Included in Backups:

  • All reptile information
  • All feeding logs
  • All shed records
  • Weight and length tracking history
  • Photo metadata (paths and captions)
  • Feeding reminder settings

What's NOT Included:

  • Actual photo files (you'll need to back these up separately)
  • User account information

Best Practices:

  • Create regular backups (weekly or monthly)
  • Store backups in multiple locations (cloud storage, external drive)
  • Test your backups occasionally by restoring to a test instance
  • Always use Merge mode unless you specifically need to replace all data
