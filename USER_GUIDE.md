## üçñ Feeding Logs

### Recording a Feeding

1. Go to reptile's profile
2. Click "Add Feeding Log"
3. Fill in details:
   - **Date**: When the feeding occurred
   - **Food Type**: What was offered (e.g., "Rat", "Mouse")
   - **Food Size**: Size of prey (e.g., "Large", "Medium")
   - **Quantity**: How many items (default: 1)
   - **Ate**: Did they eat? (Yes/No)
   - **Notes**: Any observations

4. Click "Add Log"

### Viewing Feeding History

- **From Profile**: Scroll to "Feeding Logs" section
- **Filter by Date**: Use date range filters
- **View All**: Click "View All Feeding Logs"

### Editing/Deleting Logs

1. Find the log in the list
2. Click "Edit" or "Delete"
3. Make changes or confirm deletion

## üì¶ Food Inventory Management

### Adding Stock

1. Click "üì¶ Food Inventory" in navigation
2. Click "Add Stock"
3. Fill in details:
   - **Food Type**: What you're adding (e.g., "Rats", "Mice")
   - **Food Size**: Size category (e.g., "Large", "Medium")
   - **Quantity**: How many items
   - **Unit**: Items, grams, etc.
   - **Cost per Unit**: Price per item (optional)
   - **Supplier**: Where purchased (optional)
   - **Purchase Date**: When purchased (optional)
   - **Expiry Date**: When it expires (optional)
   - **Notes**: Any additional info

4. Click "Add Stock"

**Auto-Increment Feature:**
- If item already exists, quantity is added automatically
- Example: Have 5 Large Rats, add 10 more = 15 total

### Understanding Stock Alerts

**Color Coding:**
- üü¢ **Green**: Good stock (>5 items)
- üü° **Yellow**: Low stock (1-5 items)
- üî¥ **Red**: Out of stock (0 items)

**Alert Badges:**
- "LOW STOCK" appears when quantity ‚â§ 5
- "OUT OF STOCK" appears when quantity = 0

### Adjusting Quantities

**Quick Adjustments:**
1. Find item in inventory
2. Click "+1" or "-1" buttons
3. Quantity updates immediately

**Manual Adjustments:**
1. Click on item
2. Click "Adjust Quantity"
3. Enter new quantity
4. Add note explaining change
5. Click "Update"

### Viewing Transaction History

1. Click on inventory item
2. Scroll to "Transaction History"
3. See all changes:
   - Purchases (stock added)
   - Feedings (stock used)
   - Adjustments (manual changes)
   - Waste (spoiled/damaged items)

**Transaction Details:**
- Date and time
- Type of transaction
- Quantity changed
- Notes
- Running balance

### Deleting Inventory Items

1. Click on item
2. Click "Delete"
3. Confirm deletion
4. All transaction history is preserved

## üîî Notifications

### Setting Up Email Notifications

1. Click "Notification Settings" in navigation
2. Enable "Email Notifications"
3. Enter your email address
4. Configure SMTP settings:
   - **SMTP Server**: e.g., smtp.gmail.com
   - **SMTP Port**: Usually 587
   - **Username**: Your email
   - **Password**: App-specific password

5. Click "Save Settings"

**Gmail Setup:**
1. Enable 2-factor authentication
2. Generate app-specific password
3. Use that password in settings

### Setting Up SMS Notifications

1. Click "Notification Settings"
2. Enable "SMS Notifications"
3. Enter phone number (with country code)
4. Configure Twilio:
   - **Account SID**: From Twilio dashboard
   - **Auth Token**: From Twilio dashboard
   - **From Number**: Your Twilio number

5. Click "Save Settings"

### Notification Preferences

**Reminder Time:**
- Set what time you want daily reminders
- Default: 9:00 AM

**Advance Notice:**
- Get notified 0-7 days before feeding due
- Default: 0 days (on due date only)

**Overdue Only:**
- Only notify for overdue feedings
- Useful if you don't want daily reminders

### Testing Notifications

1. Go to Notification Settings
2. Click "Send Test Email" or "Send Test SMS"
3. Check your email/phone
4. Verify notification received

## üíæ Data Management

4. Backup includes:
   - All reptiles
   - All feeding logs
   - All shed records
   - All weight/length measurements
   - All photos (paths)
   - All expenses

   - All settings

**Backup Frequency:**
- Weekly: Recommended minimum
- Before major changes
- Before updates

### Restoring Data

‚ö†Ô∏è **Warning:** This will replace or merge with existing data

## üí° Tips & Best Practices

### General Tips

**Regular Updates:**
- Log feedings immediately after they occur
- Record weight monthly
- Take photos regularly for growth comparison

**Consistent Data:**
- Use consistent food type names
- Use standard size categories
- Keep notes detailed but concise

**Organization:**
- Use tags for expenses
- Add notes to all records
- Keep receipts organized

### Feeding Tips

**Tracking Success:**
- Always mark if reptile ate or refused
- Note any unusual behavior
- Track feeding response time

**Food Inventory:**
- Reorder when stock reaches 5 items
- Check expiry dates regularly
- Rotate stock (first in, first out)

### Health Monitoring

**Weight Tracking:**
- Weigh weekly during growth phase
- Weigh monthly for adults
- Note any sudden changes

**Shed Tracking:**
- Record all sheds, even incomplete
- Note any stuck shed locations
- Track shed frequency

**Veterinary Care:**
- Keep all vet receipts
- Note medications and dosages
- Track follow-up appointments

### Expense Management

**Receipt Organization:**
- Upload receipts immediately
- Use clear descriptions
- Tag related expenses

**Budget Tracking:**
- Review monthly expense reports
- Set spending goals
- Identify cost-saving opportunities

### Data Security

**Backups:**
- Create weekly backups
- Store backups in multiple locations
- Test restore process occasionally

**Privacy:**
- Don't share sensitive data
- Use strong passwords
- Keep software updated
